# Logo Placeholder

Place your government seal image here as `logo.png`

**Recommended specifications:**
- Format: PNG with transparency
- Size: 512x512 pixels
- Design: Government seal, badge, or official emblem

**Note:** If no logo is provided, the image reference in the UI will show a broken image icon. The system will still function normally.

You can use any image editing software to create a circular seal with:
- State/server name
- Date established
- Official symbols
- Latin motto (optional)

**Temporary Solution:**
You can use any circular badge or logo temporarily until you create a custom one.
